# Workspace Analysis


Plots the workspace boundaries using the singularity-based approach
Samples joint configurations, calculates the jacobian using the **jacob0()** function in the robotics toolbox
Identifies singularities by checking the rank of the jacobian.
The identified singular configurations define the workspace boundaries

Results:
Found 9946 singular configurations (Boundaries / Internal Singularities).
Sampled 3978 non-singular configurations (Interior Workspace).
